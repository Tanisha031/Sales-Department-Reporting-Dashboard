# Sales-Department-Reporting-Dashboard
This project is a comprehensive Sales Reporting System that analyzes daily to monthly sales performance, targets, collections, KPIs, profit margins, lead conversions, and product trends. It transforms raw sales data into actionable insights to support performance evaluation, cash flow management, and strategic decision-making.
